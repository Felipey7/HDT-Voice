    What is HDT-Voice?

    HDT-Voice is a plugin for Hearthstone Deck Tracker that allows you to play the game and navigate the menus using simple voice commands.

    Requirements and Download

    The requirements for HDT-Voice are as follows:

    • HDT-Voice plugin (download here)
    • Hearthstone Deck Tracker
    • Microsoft Speech Recognition engine

    Installation and Usage

    Please see the HDT-Voice Wiki for full installation instructions and a detailed guide to the voice commands available.
